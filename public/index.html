Süper Ekonomik Blog Sitesi

Cem Celal Altınay published on 2020-03-04 included in documentation
Hugo temel statik siteler yapmak için kullanılan veritabanı olmamaksızın çalışan web site tool aracıdır. Hugo kullanımı kolay açık kaynaklı ve tamamen ücretsizdir. Bu sayede uygun fiyatlı bir web site üretmek mümkündür. Github Pages + Hugo + Google Domains = Süper ekonomik web sitesi.. Hugo Nasıl İndirilir Kurulur? Daha önce hiç windows ile kurulum yapmadığım için Linux üzerinden anlatacağım. Hugo kurulumu çok basit. https://gohugo.io/installation/linux/ Kurulum için git kullanmamız gerekiyor. Git kurulu değilse kurmanızı tavsiye ederim.
